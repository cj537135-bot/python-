Cellphone Market & User Preference Analysis
Project Overview
This project analyzes a smartphone dataset to understand how different mobile phone features influence pricing, performance, and customer satisfaction. The dataset includes information about mobile phone specifications, user ratings, and customer demographics. The objective of this project is to explore patterns in smartphone usage and identify the factors that impact user preferences and purchasing behavior.
By applying data analysis techniques, the project demonstrates how structured datasets can be processed, analyzed, and interpreted to generate meaningful insights about consumer electronics markets.
Dataset Description
The dataset contains 990 records and 22 variables describing mobile phones and user information.

Objectives
The main objectives of this project are:
Perform data cleaning and preprocessing on a real-world dataset.
Conduct exploratory data analysis (EDA) to identify trends in smartphone features and pricing.
Analyze how technical specifications influence smartphone ratings.
Understand consumer purchasing patterns using demographic data.
Identify the key factors influencing smartphone performance and user satisfaction.
Tools & Technologies Used
Python

NumPy – numerical computations
Jupyter Notebook – interactive data analysis
Matplotlib / Seaborn – data visualization
Key Analysis Performed
1. Data Cleaning
Removed missing values and inconsistencies.
Standardized numerical and categorical variables.
2. Exploratory Data Analysis
Studied relationships between price, RAM, and performance.
Analyzed battery size and screen size trends across brands.
Evaluated rating distribution across different phone models.
3. Consumer Behavior Analysis
Examined how age, occupation, and salary influence purchasing decisions.
Identified demographic patterns related to smartphone ratings.
4. Market Insights
Identified brands offering high performance at competitive prices.
Studied the correlation between hardware specifications and customer satisfaction.
Key Insights
Smartphones with higher RAM and performance scores tend to receive higher ratings.
Price is strongly correlated with internal memory and performance features.
Younger users tend to prefer high-performance devices, while older users prioritize battery life and usability.
Certain brands consistently deliver better price-to-performance value.
